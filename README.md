Healthcare Data Analytics: Analysis of Major Health Indicators at Sub-District Level
Overview

This project analyzes healthcare data at the sub-district level across India to identify regional health trends and generate meaningful insights. The project includes data cleaning, exploratory data analysis (EDA), and data visualization using Python.

Dataset
Records: 500
Original Features: 296
Features After Cleaning: 294

The dataset contains information on healthcare services, immunization, maternal health, childhood diseases, outpatient attendance, and institutional deliveries.

Tools & Technologies
Python

Project Highlights
Cleaned and preprocessed the dataset
Performed Exploratory Data Analysis (EDA)
Created multiple visualizations to identify healthcare trends
Generated insights on immunization, disease distribution, healthcare facilities, and drug availability
Key Insights
Public healthcare institutions are more prevalent than private institutions.
Rural areas have greater healthcare representation than urban areas.
Bihar and Maharashtra contribute the highest number of records.
Telangana shows strong immunization performance.
Essential drugs reported zero stock-out rates across the dataset.
