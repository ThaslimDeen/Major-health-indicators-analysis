# Healthcare Data Analytics: Analysis of Major Health Indicators at Sub-District Level

## Overview

This project analyzes healthcare data at the sub-district level across India to identify regional health trends and generate meaningful insights. The project involves data cleaning, exploratory data analysis (EDA), and data visualization using Python.

## Dataset

- **Records:** 500
- **Original Features:** 296
- **Features After Cleaning:** 294

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Highlights

- Cleaned and preprocessed the dataset
- Performed Exploratory Data Analysis (EDA)
- Created 12 visualizations using Matplotlib and Seaborn
- Generated insights on healthcare trends across different states
